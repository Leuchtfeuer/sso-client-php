Single Sign-On Client

Diese Seite simuliert einen DirectSSO Client. Der Anmeldeprozess ist direkt über die Eingabefelder oder per Single Sign-On am TYPO3 möglich

Highlights

  • Zugriff auf Anwendungen von Drittanbietern ohne zusätzliches Logon
  • Rollenbasierte Integration eines TPA in TYPO3 Content
  • Fortschrittliche 3-Layer Architektur
  • Keine Server-to-Server Kommunikation notwendig
  • Keine gemeinsame Benutzerdatenbank oder Passwortsynchronisation notwendig

Sicherheit

Für DirectSSO wurde ein Sicherheits-Framework mit den folgenden Komponenten implementiert

  • SSO Server (TYPO3 Extension)
  • SSO Agenten
  • SSO Adapter (TPA) zur Integration

Details finden Sie unter DirectSSO whitepaper
